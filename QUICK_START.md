# Quick Start Guide

Get up and running in 3 steps!

## Step 1: Set Your API Key

```bash
export GEMINI_API_KEY='your-api-key-here'
```

Windows (PowerShell):
```powershell
$env:GEMINI_API_KEY='your-api-key-here'
```

## Step 2: Set Up Directories

```bash
python setup_directories.py
```

This creates the directory structure and optionally migrates your existing files.

## Step 3: Run

```bash
python main.py
```

Edit `main.py` first to specify your input files if needed.
